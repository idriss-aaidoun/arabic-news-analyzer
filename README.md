# Arabic News Analyzer

Arabic Sentiment Analysis Platform built with:

- Python
- Streamlit
- Hugging Face Transformers
- AraBERT / CAMeLBERT

## Features

- Arabic Sentiment Analysis
- Keyword Extraction
- Dataset Dashboard
- Model Comparison

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```
